Bonjour jeune invocateur.

Vous êtes rentrés dans une dangereuse période.

Deux grands mondes se sont réunis mais se disputent ce monde.

Vous êtes l'élus et devez rejoindre l'un des camps, vos choix décideront de l'avenir de ce monde.

Afin de vous lancer, dans l'aventure veuillez cliquer sur echec_ouverture afin vous immerger dans cet univers.

Cliquez sur Start et choisissez votre clan en cliquant sur la banniere de votre équipe préféré.

Le but est de prendre le roi ennemi, mais faites attention il est protégé autant que le votre.

La reine peut aller sur les chemins droits ou en diagonales.

La tour que sur les chemins droits.

Le fou sur les chemins en diagonales.

Les pions peuvent avance de deux cases leurs premier coup, sinon ils n'avancent que d'une case. Si une piece ennemi est mise en diagonale
il peut la prendre. Si il traverse le terrain, il peut evoluer en autre piece autre que le roi.

Si le roi est isolé et tous ses chemins sont condamnés par l ennemi, vous perdez ce combat.
